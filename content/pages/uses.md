# uses

stuff i use _(inspired by [ovyerus](https://ovyerus.com/uses/) and
[natalie](https://natalie.sh/uses))_

## editor and terminal

- I use [neovim](https://neovim.io) as my text editor/IDE, and I've been happily
  daily driving it since 2021. I use vim.pack as my package manager, and my
  [config](https://github.com/ptdewey/nvim) is written in
  [fennel](fennel-lang.org).
- I use [WezTerm](https://wezterm.org/) as my terminal, which serves as a
  launcher for my real terminal, [tmux](https://github.com/tmux/tmux). Despite
  the fact that I have done very little configuration of it, I like that WezTerm
  is configured in Lua.
- My programming font is my own
  [custom build](https://github.com/ptdewey/dotfiles/tree/main/fonts/patricks-iosevka-patched)
  of [Iosevka](https://typeof.net/Iosevka/).
- My editor and terminal theme is
  [darkearth](https://github.com/ptdewey/darkearth-nvim), a dark and earthy
  theme I created (I usually use the light version, lightearth, now though).
- I use Neovim for note taking on my computers, and Obsidian when I'm on my
  phone.
- I've also been getting really into using and customizing
  [pi coding agent](https://pi.dev).

## linux

- I run NixOS (a distro that lets me
  [declaratively configure](https://github.com/ptdewey/nixos) my entire system)
  on all of my systems.
- I have been using Niri as my main window manager for a couple of years, but I
  am exploring [river](https://codeberg.org/river/river) on my laptop.
- My [dotfiles](https://github.com/ptdewey/dotfiles) are mostly just shell
  scripts, I don't do much ricing and I just use waybar and a (badly)
  home-rolled quickshell bar.

## keyboards

- I use a [Cyboard Imprint](https://cyboard.digital) with TTC Frozen Silent
  switches as my main keyboard.
- When traveling, I generally use a
  [Piantor Pro](https://shop.beekeeb.com/products/pre-soldered-piantor-split-keyboard)
  with Ambient Silent Twilight switches.
- I also own a
  [Keebio Iris](https://keeb.io/collections/iris-split-ergonomic-keyboard) with
  Bsun Silent Sakura switches which I also use somewhat regularly.

## computers

  gpu.
